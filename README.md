# 🎨 Color Switcher

A simple and interactive **Color Switcher** built using **HTML**, **CSS**, and **JavaScript**. Click on any colored button to instantly change the background color of the webpage.

## 🚀 Features

* 🎨 Switch the background color with a single click
* ⚡ Instant color updates
* 📱 Simple and responsive layout
* 🧩 Built using vanilla JavaScript (no frameworks)

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
